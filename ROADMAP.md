# Roadmap

Quick Memo Applet 当前目标是保持轻量、稳定、可打包的 Windows-first 桌面便签工具。路线图会随着实际使用反馈调整。

## v0.1.x 发布完善

- 补充 README 截图和使用示例
- 完善 Windows Release 打包检查清单
- 验证解压后直接运行的干净机器体验
- 记录常见快捷键冲突和部署问题

## v0.2 用户体验

- 继续整理 Dashboard 设置分组和状态反馈
- 优化首次启动说明和托盘提示
- 改进记录完成/删除的可发现性
- 评估更清晰的导入导出反馈

## v0.3 工程化

- 增加 Windows CI 构建检查
- 添加基础 smoke test 或打包脚本
- 评估更正式的 Release 附件生成流程

## 暂不计划

- 云同步或账号系统
- QML 重写
- 跨平台优先支持
- 重型插件系统
