# Roadmap

Quick Memo Applet 当前目标是保持轻量、稳定、可打包的 Windows-first 桌面便签工具。v1.0.0 之后，路线图会继续围绕可靠发布、数据安全和小范围体验改进推进。

## v1.x 稳定化

- 补充 README 截图和短动图。
- 完善手动验收清单，覆盖托盘、快捷键、输入、便签窗口、导入导出和开机自启。
- 继续观察安装包、zip 包和干净 Windows 环境运行反馈。
- 收集常见快捷键冲突和部署问题。

## v1.x 工程化

- 评估 Windows CI 构建检查。
- 评估自动化 smoke test。
- 继续收敛发布流程，减少手工步骤。

## v1.x 体验改进

- 改进记录完成/删除的可发现性。
- 补充更清晰的空状态和错误反馈。
- 评估便签窗口的小范围交互 polish。

## 暂不计划

- 云同步或账号系统
- QML 重写
- 跨平台优先支持
- 重型插件系统
